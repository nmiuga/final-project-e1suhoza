## Restaurant Journal

A personal iOS app for logging and remembering restaurants you've visited. Track your meals, rate your experiences, add photos, and find restaurants near you — all in one place.

## Features

- **Log restaurants** — save the name, cuisine, location, and visit date
- **Rate your experience** — 1 to 5 star rating system
- **Add photos** — attach up to 6 photos per restaurant from your photo library
- **Search** — filter by name, cuisine, or notes
- **Sort** — organize by most recent, highest rated, or alphabetically
- **Pick Random** — can't decide where to eat? Let the app pick for you
- **Near Me** — filter restaurants by your current city using GPS

## Setup Instructions

1. Clone the repository:
```bash
   git clone https://github.com/your-username/final-project.git
```
2. Open `Final Project.xcodeproj` in Xcode
3. Select a simulator or connected device (iPhone recommended)
4. Press **Command + R** to build and run

> No third-party dependencies or API keys required — the app runs entirely with native Apple frameworks.
